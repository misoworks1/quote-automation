# 견적서 자동화 시스템 (Google Apps Script)

Google Sheets 표준 견적서 템플릿에 데이터를 입력하고 PDF로 변환하는 자동화 도구

---

## 개발 환경

| 항목 | 내용 |
|------|------|
| 플랫폼 | Google Apps Script (V8 런타임) |
| 에디터 | Google Apps Script Editor (내장) 또는 clasp CLI |
| 언어 | JavaScript (ES6+) |
| 의존성 | Google Sheets API, Google Drive API (Apps Script 내장) |
| 버전관리 | Git + GitHub |
| 로컬 개발 (선택) | Node.js + [clasp](https://github.com/google/clasp) |

---

### 구성 요소

1. **견적서 템플릿 (Google Sheets)**
   - 표준 견적서 양식 (발신자 정보 고정, 수신자/품목 동적 입력)
   - 공급가액, 세액, 합계 자동 계산 수식 포함

2. **입력 UI (HTML 사이드바)**
   - 수신자 정보: 업체명, 담당자, 연락처, 이메일
   - 품목 정보: 품명, 수량, 단가 (행 동적 추가/삭제)
   - 견적일자, 유효기간
   - [생성] / [PDF 다운로드] 버튼

3. **Apps Script 백엔드 (Code.gs)**
   - 사이드바 UI 서빙
   - 시트 데이터 쓰기/읽기
   - PDF 변환 및 다운로드 링크 생성

## 배포 환경

| 항목 | 내용 |
|------|------|
| 실행 환경 | Google Apps Script (클라우드, 서버리스) |
| 호스팅 | Google 인프라 (별도 서버 불필요) |
| 접근 방식 | Google Sheets 메뉴 → 사이드바 실행 |
| 인증 | Google 계정 OAuth (스크립트 최초 실행 시 권한 승인) |
| 배포 방법 | Apps Script 에디터에서 직접 저장 또는 clasp push |

### 배포 절차

1. Google Sheets에서 `확장 프로그램 > Apps Script` 열기
2. `Code.gs`, `Sidebar.html` 파일 붙여넣기
3. 스크립트 저장 후 시트 새로고침
4. 메뉴 `견적서 > 견적서 작성` 클릭하여 사이드바 실행
5. 최초 실행 시 Google 권한 승인
